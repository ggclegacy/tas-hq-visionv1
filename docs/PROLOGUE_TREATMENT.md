Status: BUILD 001 production treatment. Neil's subjective visual acceptance is pending.

Exact timed captions and a complete silent path are implemented; browser text-to-speech and runtime voice generation are prohibited.

Essential retains architecture, marks, semantic copy, and premium still lighting. Enhanced is the older-Mac default and adds bounded light/glass depth. Cinematic adds foreground aperture and reflection finish. Reduced motion uses stable framing and short light/opacity changes while preserving all meaning.
